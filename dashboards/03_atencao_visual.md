# Dashboard 03 – Atenção Visual aos Mobiliários Urbanos

## Objetivo de Negócio
Identificar quais elementos do mobiliário urbano geram maior destaque visual para o usuário.

## Perguntas Respondidas
- Os usuários percebem os mobiliários urbanos?
- Quais elementos chamam mais atenção?
- A atenção está concentrada em poucos ativos?

## Valor Analítico
Apoia a priorização de investimentos em elementos com maior retorno perceptivo e impacto na experiência do usuário.
